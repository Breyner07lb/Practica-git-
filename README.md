# Practica-git- 



Este proyecto es una aplicación ficticia diseñada para facilitar la administración y monitoreo de servidores Linux.

Su objetivo es permitir a los administradores gestionar recursos, revisar el estado del sistema y automatizar tareas de mantenimiento de forma **eficiente**.



## Instalación



1. Clonar el repositorio.

2. Ingresar a la carpeta del proyecto.

3. Instalar las dependencias necesarias.

4. Ejecutar la aplicación.



## Tecnologías utilizadas



- Linux

- Git

- GitHub

- Bash

- Markdown
